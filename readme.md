# resume.spilth.org

My resume in Markdown format and a script that converts it to HTML and PDF.

## Installation & Generation

1. Install [Homebrew](http://mxcl.github.com/homebrew/)
1. `brew install htmldoc`
1. `bundle`
1. `rake`
